# ft_containers

## Description
Reimplementation of STL containers with C++98.

## Requirement

- gcc or Clang
- GNU Make

## Features
Implement a series of containers: 
- vector
- stack
- map

## Test
```
git clone https://github.com/ryo-manba/ft_containers
cd ft_containers

make test  # Run Unit-test
make bench # Run Benchmark-test
```
